    - Mídia

        -Música
            -Contém:
                -Duração

                    -Contém:
                        .Getters 
                        .Setters
        -Filme

                -Duração:

                        -Getters
                        -Setters

        -Contém:
            -Nome
            -Data de Nascimento
            -Premiações
            -Gênero(MAS/FEM/IND)
                -Contém:
                    .Getters
                    .Setters
        -Contém:
            -Ator
                -Contém:
                    -Formação(Teatro/Cinema/Outro)
            -Músico
                -Contém:
                    -Atuação(Instrumento ou vocal)
    -Biblioteca de Mídias
        -Contém:
            -Mídias

                    .Setters
